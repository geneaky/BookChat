== 공통 HTTTP Status Code

|===
| Status Code | Common Mean |    Usage
|     200     |   Success   |요청 정상 처리
|     400     | Bad Request |요청이 부적절한 경우
|     401     | UnAuthorized|인증되지 않은 사용자
|     403     |  Forbidden  |권한이 없거나 차단된 사용자
|===
